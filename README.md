# 🏫 Smart Time Table

Սա դպրոցական դասացուցակների խելացի կառավարման համակարգ է՝ ստեղծված **Python**-ով և **Streamlit**-ով։ 

Այն օգնում է արագ և հեշտությամբ ստեղծել դասացուցակներ՝ բացառելով մարդկային սխալները և կոնֆլիկտները։

---

### ✨ Ի՞նչ կարող է անել այս համակարգը

* **📊 Վահանակ**՝ Տվյալների վիճակագրություն և գեղեցիկ դիզայն
* **📚 Առարկաներ**՝ Առարկաների բազայի կառավարում
* **👩‍🏫 Ուսուցիչներ**՝ Ուսուցիչների և ժամերի բաշխում
* **🏫 Դասարաններ**՝ Դասարանների կարգավորումներ
* **🚀 Ավտոմատ Գեներացում**՝ Դասացուցակի ստեղծում մեկ կլիկով
* **🤖 AI Օգնական**՝ Վերլուծություն Gemini-ի միջոցով

---

### 🚀 Ինչպե՞ս է աշխատում Խելացի Գեներացումը

Համակարգի հիմքում ընկած ալգորիթմը վերլուծում է քո մուտքագրած բոլոր տվյալները և ավտոմատ կերպով ստեղծում դասացուցակներ՝ բացառելով հետևյալ խնդիրները․

* **👥 Ուսուցիչների համընկնումներ** — Նույն ուսուցիչը չի կարող միաժամանակ լինել երկու տարբեր դասարաններում։
* **⏱️ Ժամաքանակի գերազանցում** — Յուրաքանչյուր ուսուցիչ ստանում է ճիշտ այնքան ժամ, որքան նախատեսված է։
* **🧠 Առարկաների բարդության հաշվարկ (Complexity Index)** — Համակարգը հաշվի է առնում աշակերտների օրական մտավոր ծանրաբեռնվածության կորը։ Բարձր կենտրոնացում պահանջող ծանր առարկաները (օրինակ՝ Մաթեմատիկա, Ֆիզիկա) ավտոմատ կերպով դասավորվում են օրվա առաջին կեսին, իսկ թեթև կամ ստեղծագործական առարկաները՝ վերջին ժամերին։
* **🛑 Դատարկ պատուհանների (Windows) կրճատում** — Ալգորիթմը փորձում է հավաքել դասերը հնարավորինս հոծ՝ առանց անիմաստ սպասումների։
